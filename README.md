# Mexican Restaurant - ASP.NET Core MVC 

A beginner-friendly Mexican Restaurant web application built with **ASP.NET Core MVC.**

## Learning Objectives
The main goal of this project was to master:
- **MVC Architecture:** Separation of concerns between Models, Views, and Controllers.
- **Entity Framework Core (Code First):** Designing the database using C# classes and managing migrations.
- **SQL Server Integration:** Persisting data for menu items, ingredients, and categories.
- **Razor Syntax:** Building dynamic, data-driven user interfaces.
- **CRUD Operations:** Implementing Create, Read, Update, and Delete functionality for a restaurant menu.

## Tech Stack
* **Framework:** .NET 10.0 (ASP.NET Core MVC)
* **Database:** Microsoft SQL Server
* **ORM:** Entity Framework Core
* **Frontend:** Razor Views, Bootstrap 5, CSS3
* **Tools:** Visual Studio 

## Key Features
- **Menu Management:** Browse Mexican cuisine categories and view detailed ingredient lists.
- **Dynamic Views:** Responsive design using Bootstrap for a seamless mobile/desktop experience.
- **Data Persistence:** Fully integrated database to store restaurant data.
- **ERD Implementation:** Database structure based on a real-world Entity Relationship Diagram.
